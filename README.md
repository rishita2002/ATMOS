## Installation

1. Clone the repository:
   ```
   git clone https://github.com/rishita2002/ATMOS
   ```

2. Navigate to project directory and install requirements.

   ```
   cd ATMOS
   pip install -r requirements.txt
   ```

3. Running app.py
   ```
   python3 app.py
   ```

4. Grab the link for local sever.
